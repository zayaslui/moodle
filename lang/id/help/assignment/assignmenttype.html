Jenis Tugas

Ada beberapa perbedaan jenis tugas

Kegiatan Offline - ini sangat berguna bila tugas disampaikan diluar dari Moodle. Siswa dapat melihat penjelasan dari tugas tersebut, tetapi tidak dapat meng-upload file atau apapun. Pengajar dapat memberi nilai semua siswa lebih dulu, dan siswa akan melihat pengumuman mengenai nilai mereka.

Upload file tunggal - jenis tugas yang seperti ini membolehkan semua siswa untuk mengirimkan (upload) file tunggal, sembarang jenis. Ini mungkin adalah dokumen dari pengolah kata, ata sebuah gambar, situs web yang terkompres, atau apa saja yang Anda mintakan mereka untuk dikirimkan. Pengajar dapat memberi nilai pada tugas yang dikirimkan secara online.
