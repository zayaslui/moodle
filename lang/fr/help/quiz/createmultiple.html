Créer plusieurs questions aléatoires

Cette commande vous permet de créer un jeu de questions tirées au hasard dans la catégorie que vous aurez choisie. Cet ensemble peut être ajouté à un test. Vous devez donner le nombre de questions à inclure dans cet ensemble.

Vous pouvez ainsi créer un test qui serait entièrement généré au hasard et qui serait différent pour chaque étudiant. Plus le nombre de questions dans la catégorie est grand, plus les tests générés seront différents les uns des autres.

Vous pouvez également inclure cet ensemble de questions aléatoires dans un test de façon à ce qu'une partie seulement du test soit aléatoire.

Il est important que les questions servant à établir un test construit aléatoirement soient de même niveau.
