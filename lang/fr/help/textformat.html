Formatage d'un texte

Lorsque vous saisissez du texte dans Moodle, vous pouvez choisir entre plusieurs formats en fonction de votre expertise et du type de navigateur utilisé.

La plupart du temps, le format par défaut produira les résultats attendus..

1. Format Moodle

    Ce format est préférable à l'éditeur WYSIWYG intégré lorsque vous entrez du texte dans un formulaire standard sur une page Web. Écrivez normalement votre texte comme si c'était un courriel.

    Moodle effectuera certaines transformations lorsque vous sauvegarderez votre texte.

    Moodle ajoutera automatiquement des balises HTML nécessaires pour que :

    Les adresses URL, par exemple http://moodle.org/ et www.etoile-des-enfants.ch seront transformées en liens hypertextes.

    Les changements de ligne et les lignes vides seront conservées.

    Les binettes, par exemple :-), seront remplacées par leurs équivalents graphiques.

    Vous pouvez insérer du code HTML, qui sera interprété correctement.

    Informations supplémentaires sur le format Moodle

2. Format HTML

    Ce format est le format des pages Web. Il est constitué de texte et de balises HTML. Il est utilisé par défaut par l'éditeur WYSIWYG intégré. Les boutons de la barre d'outils produisent automatiquement le code HTML pour vous.

    Si vous n'utilisez pas l'éditeur WYSIWYG intégré, vous pouvez saisir directement le code HTML pour obtenir le résultat désiré.

    Il n'y a aucun formatage automatique comme dans le format automatique de Moodle.

3. Format texte

    Utilisez ce format lorsque vous avez beaucoup de code ou de balises HTML à écrire. Seuls les espaces et les retours à la ligne seront produits automatiquement. Tout le reste ne sera pas formaté.

4. Format Wiki

    Ce format utilise pour le formatage du texte des codes simplifiés ; il est connu sous le nom de Texte Wiki.

    Les textes Wiki demeurent très lisibles malgré les codes employés pour les formater. Ces codes seront automatiquement convertis en balises HTML pour la génération des listes, des entêtes et autres formatages complexes.
