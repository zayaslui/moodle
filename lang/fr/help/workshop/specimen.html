Exemple de formulaire d'évaluation

Cette page montre un exemple de formulaire qui sera utilisé pour l'évaluation de votre travail. Ce formulaire sera employé par :

  • l'enseignant, s'il évalue votre travail ;
  • vous-même, si l'enseignant demande une auto-évaluation de votre travail ;
  • un certain nombre d'étudiants, si l'enseignant a décidé d'utiliser l'évaluation par les pairs.

Veuillez remarquer que bien que vous puissiez changer les notes sur ce formulaire, ces modifications ne seront pas enregistrées. Il ne s'agit que d'un exemple de formulaire.
