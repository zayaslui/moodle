Programmes externes

Un programme externe est une ressource externe qui peut accepter des données en provenance de Moodle.

Les données, incorporées à l'adresse URL, sont transmises par http_get et comprend :

  • extern_nav : l'adresse URL à partir de laquelle la ressource externe est appelée
  • extern_usr : le nom de l'utilisateur courant
  • extern_nam : le nom complet de l'utilisateur courant
  • extern_tim : l'heure actuelle
  • extern_pwd : un mot de passe unique

Ces informations et ce procédé seront modifiés sous peu.
