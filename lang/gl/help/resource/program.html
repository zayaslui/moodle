Programas como materiais

Un Programa é un recurso externo que pode aceptar datos enviados desde Moodle.

Os datos pásanse vía HTTP GET (como parte da URL) e inclúen:

  • extern_nav: a URL desde onde se está chamando ao recurso
  • extern_usr: o nome de usuario do usuario/a actual
  • extern_nam: o nome completo do usuario/a actual
  • extern_tim: a data actual
  • extern_pwd: un único contrasinal
