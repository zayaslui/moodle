Como dar formato ao texto

Cando escribe texto ten dúas formas de darlle formato, basicamente depende do navegador que utilice.

Normalmente Vd. non necesita preocuparse por isto, todo funcionará como é de esperar.

1. Formato automático

    Este formato é o mellor para a comunicación tradicional, teclee normalmente o seu texto como se redactara un correo electrónico.

    Cando garde o seu texto automaticamente se farán os seguintes cambios:

    Enderezos de Internet http://enlaceacademico.com ou incluso www.enlaceacademico.com convertiranse en enlaces.

    Manteranse os seus saltos de liña, e as liñas en branco indicarán un novo parágrafo.

    Emoticonas como :-), cambiaranse polos seus equivalentes gráficos.

    Tamén pode incrustar código HTML.

    Máis información sobre o formato automático.

2. Formato HTML

    Este formato asume que a mensaxe está escrita en HTML puro. Se está empregando o editor de HTML, este será o formato por defecto, o editor creará o código por vostede.

    Incluso se está empregando o editor HTML, pode colocar código directamente no texto.

    A diferencia do formato automático, neste modo non se procesará nada.
