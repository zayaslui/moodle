如何有效的發問

讓別人認真思考一個問題最好的方式便是詢問他對問題的觀點. 一個好的問題可以協助將相關訊息充分組合. 評估現有建議或方案的可行性, 甚至引導新的觀點.

所謂[蘇格拉底式問題]便是藉由問題協助受問者進行學習.

蘇格拉底式問法要求提問能夠讓對方感覺有意義, 具有建設性而非衝突性的方式.

以下是一些問問題方式的範例:

確定討論內容的問法

  • 當你說 ... 時,你心中真正的意思是...?
  • 你最主要的論點是...?
  • ... 和 ... 兩者間有何關聯?
  • 你能夠用另一種方式說明你的觀點嗎?
  • 讓我確定是否我已經體會你的想法, 你是否認為 ... 或者 ...?
  • 你現在所談論的和目前的問題/討論/議題有何相關性?
  • [某甲], 是否你能用你自己的話描述 [某乙]所剛描述的話? ... [某乙], 是否[某甲]所說的是你心中的意思?
  • 你是否能夠舉例說明?
  • ... 是否和你說的情形類似?

確定對方立場(假設觀點)的問法

  • 對這件事你的立場(假設)為何?
  • 你認為 某甲的立場為何?
  • 是否有另外的可能(想法)?
  • 你似乎認為 .... 是否我對你的理解是正確的?
  • 似乎你的推論都是基於 ... 的想法. 為何你會認為情形是 ... 而不是 ...,有何判斷依據?
  • 你似乎認為 .... 你如何確認哪樣的想法是沒有問題的呢?
  • 是否情形總是...? 你認為是否有共通的原因呢?
  • 為何有人會有... 的想法呢?

探詢理由或證據的問法

  • 你可以解釋(說明)你的理由嗎?
  • 你的理由和這個事件的關聯性為何?
  • 哪樣的證據是否有值得懷疑之處?
  • 還有誰可以證實哪是對的?
  • 當某甲說 ... 你會如何回答呢?
  • 是否還有其他人可以提供你得觀點的有力證據呢?
  • 為何你會哪樣認為(如此推論)?
  • 我們如何去驗證這(事件/論點)是否是正確的?

探詢對問題觀點/想法的問法

  • 你這樣說的道理是基於...?
  • 當你說 ..., 你是否認為會...?
  • 但是, 如果發現了...的現象, 你認為會造成...的結果? 為什麼?
  • 哪會造成怎樣的效應?
  • 在...情形下,哪些事情一定會發生或有哪些事情可能會發生?
  • 有沒有其他的可能性呢?
  • 假如...和... 都是真的, 哪麼還有哪些一定也是真的?
  • If we say that ... is ethical, how about ...?

Questions that probe implications and consequences

  • How can we find out?
  • What does this question assume?
  • Would ... ask this question differently?
  • How could someone settle this question?
  • Can we break this question down at all?
  • Is this question clear? Do we understand it?
  • Is this question easy or hard to answer? Why?
  • Do we all agree that this is the question?
  • To answer this question, what other questions must we answer first?
  • How would ... state the issue?
  • Why is this issue important?
  • Is this the most important question, or is there an underlying question?
  • Can you see how this might relate to ...__?
Questions adapted from Paul, R. (1993). Critical Thinking: How To Prepare Students for a Rapidly Changing World: Foundation for Critical Thinking, Santa Rosa, CA.
