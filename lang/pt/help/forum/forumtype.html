Tipos de fórum

Existem vários tipos diferentes de fórum para escolher:

Uma única discussão simples - consiste num tópico único, tudo numa página. Útil para discussões curtas, bem focadas.

Fórum genérico para uso geral - é um fórum aberto onde qualquer pessoa pode iniciar um novo tópico de discussão em qualquer momento. Esta é a melhor opção para uso geral.

Cada pessoa envia uma discussão - cada pessoa pode iniciar unicamente um novo tópico de discussão (no entanto todos podem participar na discussão). Este tipo é útil quando quiser que cada aluno tenha que propor uma discussão, por exemplo, as suas reflexões sobre o tópico da semana, e que todos os outros alunos respondam.

(Futuras versões de Moodle terão mais outros tipos de fórum)
