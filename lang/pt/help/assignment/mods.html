Trabalhos

    Os trabalhos permitem ao docente atribuir uma tarefa que exige que os alunos preparem algum conteúdo em forma digital (em qualquer formato) que logo vão submeter usando o sistema de envio de ficheiros para o servidor. O tipo típico de trabalho inclui ensaios, projectos, relatórios, etc. Este módulo inclui funcionalidade para atribuir notas.
