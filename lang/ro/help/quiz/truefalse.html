Întrebãri Adevãrat/Fals

Ca rãspuns la o întrebare (care poate sã includã o imagine) cel examinat alege una dintre cele douã variante de rãspuns: True sau False.

Dacã feedback-ul este activat, atunci mesajul de feedback este arãtat celui examinat imediat sub rãspuns. De exemplu, dacã rãspunsul corect este "False", dar el rãspunde "True" (greºind) atunci feedback-ul de la "True" este arãtat.
