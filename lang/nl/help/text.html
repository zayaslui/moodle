Hulp bij het schrijven van teksten

Tekst schrijven in Moodle werkt een beetje zoals je het zou verwachten, maar je hebt ook nog de mogelijkheid om "smilies", webadressen en sommige HTML tags op te nemen in je tekst.

Smilies (emoticons)

    Om deze kleine icoontjes in je tekst op te nemen, hoef je alleen maar de overeenkomende code te typen. Deze codes zijn zelf als kleine tekeningen als je je hoofd schuin naar links houdt terwijl je ze bekijkt.

URLs

    Elk "woord" dat begint met www. of met http:// zal automatisch omgezet worden in een klikbare hyperlink.

    Bijvoorbeeld: www.yahoo.com of http://curtin.edu

HTML tags

    Je kunt een beperkte set HTML tags gebruiken om je tekst van opmaak te voorzien.
    HTML tagsWeergave
    <B> bold </B> vette tekst
    <I> italic </I> schuine tekst
    <U> underline </U> onderlijnde tekst
    <FONT SIZE=1> small </FONT> klein
    <FONT SIZE=4> large </FONT> groot
    <FONT COLOR=green> voorbeeld </FONT> voorbeeld
