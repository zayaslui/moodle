Programma bronnen

Een programma bron is een externe bron die gegevens kan ontvangen vanuit Moodle.

De gegevens worden doorgegeven via HTTP GET (als deel van de URL) en bestaan uit:

  • extern_nav: de URL van waaruit de bron wordt aangeroepen
  • extern_usr: de gebruikersnaam van de huidige gebruiker
  • extern_nam: de volledige naam van de huidige gebruiker
  • extern_tim: de huidige tijd
  • extern_pwd: een uniek wachtwoord

Deze functie zal in de nabije toekomst nog herbekeken worden.
