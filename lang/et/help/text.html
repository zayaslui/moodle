Abi teksti kirjutamisel
Teksti kirjutamine Moodle’is toimib üsna ootuspäraselt, kuid sul on võimalus teksti sisse lisada ka emotikone, veebiaadresse, ja mõnesid HTML manuseid.

URL e. veebiaadressid

    Mistahes „sõna“, mis algab www. või http:// muudetakse automaatselt klikitavaks lingiks.
    Näiteks : www.neti.ee või http://ametikool.ee

HTML tag-id
Võid kasutada piiratud komplekti HTML tag-e, et oma tekstis midagi rõhutada.

    HTML tag-id Tulemus
    <B> rasvane </B> rasvane tekst
    <I> kursiiv </I> Kursiivis tekst
    <U> alla joonitud </U> Allajoonitud tekst
    <FONT SIZE=1> väike </FONT> väike
    <FONT SIZE=4> suur </FONT> suur
    <FONT COLOR green> näide </FONT> näide
    <UL><LI>üks <LI>kaks </UL>
