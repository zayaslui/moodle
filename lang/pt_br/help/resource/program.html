Programa

Um Programa é um recurso externo que aceita dados enviados pela plataforma.

Os dados são enviados via HTTP GET (como parte da URL) e incluem:

  • extern_nav: a URL que indica o programa externo
  • extern_usr: o nome de usuário
  • extern_nam: o nome completo do usuário
  • extern_tim: a ora atual
  • extern_pwd: uma senha unica

Tudo isto será revisto em um futuro próximo.
