Κλίμακες

Οι καθηγητές μπορούν να δημιουργήσουν νέες κλίμακες για να τις χρησιμοποιήσουν σε ένα μάθημα για οποιαδήποτε δραστηριότητα βαθμολόγησης.

Το όνομα της κλίμακας πρέπει να είναι μια φράση που να το προσδιορίζει καθαρά: αυτό θα εμφανίζεται στις λίστες επιλογής κλίμακας, όπως και στα κουμπιά βοήθειας ευαίσθητου περιεχομένου.

Η ίδια η κλίμακα καθορίζεται από μια ταξινομημένη λίστα τιμών, από τις αρνητικές ως τις θετικές, χωρισμένη με κόμματα. Για παράδειγμα:

Απογοητευτικός, Όχι αρκετά καλός, Μέτριος, Καλός, Πολύ Καλός, Άριστος!

Οι κλίμακες πρέπει επίσης να περιλαμβάνουν μια καλή περιγραφή του τι σημαίνει και πώς αναμένεται να χρησιμοποιηθεί. Αυτή η περιγραφή θα εμφανίζεται στις σελίδες βοήθειας για καθηγητές και μαθητές.

Τέλος, μπορεί να υπάρχουν μια ή περισσότερες "Σταθερές" κλίμακες καθορισμένες στο site σου από το διαχειριστή του συστήματος. Αυτές θα είναι διαθέσιμες σε όλα τα μαθήματα.
