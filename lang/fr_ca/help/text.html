URLs

    Toute expression commençant par www. or http:// sera automatiquement convertie en hyperlien.

    Par exemple : www.yahoo.com or http://curtin.edu

Balises HTML

    Vous pouvez utiliser un sous-ensemble de balises HTML pour agrémenter vos textes. Toutes les balise doivent être en minuscules

Balises HTML

Résultats

<b> gras </b>

bold text

<i> italique </i>

italic text

<u> souligné </u>

underlined text

<pr> paragraphe</pr>
 (Changement de ligne avec espace entre les paragraphes)
<br />
 (Changement de ligne sans espace entre les paragraphes)

<font size=1> petit </font>

small

<font size=4> grand </FONT>

large

<font color=green> exemple </FONT>

exemple
