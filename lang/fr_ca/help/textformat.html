Formatage d'un texte

Lorsque vous saisissez du texte dans Moodle, vous pouvez choisir entre plusieurs formats en fonction de votre expertise ou du type de fureteur utilisé.

La plupart du temps, le format par défaut devrait produire les résultats attendus..

1. Format automatique de Moodle

    Ce format est préférable lorsque vous entrez du texte dans un formulaire standard sur une page Web (au lieu de l'éditeur intégré HTML). Écrivez normalement votre texte comme si c'était un courriel.

    Moodle effectuera certaines transformations lorsque vous sauvegarderez votre texte.

    Moodle ajoutera automatiquement des balises HTML nécessaires pour que :

    Les adresses URL par exemple http://yahoo.com et www.yahoo.com seront transformées en hyperlien.

    Les changements de ligne et les lignes vides seront conservés

    Les binettes par exemple :-) seront remplacées par leurs équivalents graphiques

    Le code HTML sera conservé

    Informations supplémentaires sur le format automatique de Moodle

2. Format HTML

    Ce format est le format des pages Web qui est constitué de texte et de balises HTML. Il est utilisé par défaut par l'éditeur intégré. Les icônes de la barre de commandes vous permettent de produire automatiquement les balises. Toutefois, il vous est possible d'entrer directement les balises désirées.

    Il n'y a aucun formatage automatique comme dans le format automatique de Moodle.

    Even if you are not using the Richtext HTML editor, you can use HTML code in your text and it should come out exactly as you intended.

    Informations supplémentaires sur l'éditeur intégré HTML

3. Format texte

    Utilisez ce format lorsque vous avez beaucoup de code ou de balises HTML à écrire. Seuls les binettes et les changements de ligne seront produits automatiquement.

4. Format de texte Wiki

    Ce format utilise des codes simplifiés pour le formatage du texte et est connu sous le nom de Texte Wiki.

    Les textes Wiki demeurent très lisibles malgré les codes employés pour les formater. Ces codes seront automatiquement convertis en balises HTML pour la génération des listes, des entêtes et autres formatages complexes.
