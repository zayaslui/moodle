Ocultar les notes

Es pot usar aquesta opció quan hi ha d'haver consens entre els estudiants sobre una avaluació. El valor per defecte mostra a l'estudiant el treball del qual s'avalua els comentaris i les avaluacions dels companys. Això pot portar a més discussions que quan l'opció està activada i es poden veure les avaluacions però no les notes.

Si l'opció és amagar les notes en les avaluacions dels companys les puntuacions es revelen un cop s'ha assolit el consens. Aquest consens, naturalment, s'haurà aconseguit només sobre els comentaris. Si aquests no encaixen raonablement amb les puntuacions aleshores l'estudiant el treball del qual s'avalua pot acudir al professorat.
