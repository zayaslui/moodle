Ús del Wiki


El Wiki és una plataforma per a treballar plegats en la construcció de pàgines web. Els principis són simples.

  • Algú escriu un text i el desa.
  • Més tard, una altra persona llegeix aquest text i pensa que es podrien fer certs canvis i afegir algunes coses. Fa clic al botó "Edita" i modifica la pàgina.
  • Quan desa la nova versió, aquesta està disponible per a tothom.


Amb un wiki, tambés es pot afegir fàcilment una pàgina nova o un enllaç a pàgines existents.

  • Un ellaç es crea per mitjà d'un MotWiki. Aix_ és un mot amb almenys dues lletres majúscules. Si ja existia una pàgina amb aquest nom, es crearà un enllaç automàticament. Si no, es visualitza un signe d'interrogació.
  • Fent clic en aquest interrogant es crea una pàgina nova en blanc amb aquest nom que es pot editar.



Regles de formatatge del wiki

Paràgrafs

  • separeu els paràgrafs per mitjà de línies en blanc
  • uilitzeu tres signes de percentatge %%% per forçar un salt de línia
  • si poseu espais o tabuladors al començament d'una línia, el text se sagnarà

!! Encapçalaments

  • utilitzeu un signe d'admiració ! al començament de la línia per crear un encapçalament de mida petita
  • !! més gran
  • !!! la mida més gran

Estils

  • si voleu emfasitzar el text, poseu-lo entre cometes simples '' (generalment es visualitza en lletra itàlica)
  • la negreta s'aconsegueix amb dos caràcters de subratllat __ (o amb asteriscos **)
  • poseu el text gran entre caràcters ##
  • podeu aconseguir text petit usant "µµ"
  • una pòlissa tipus màquina d'escriure s'obté posant el text entre dos iguals ==

Llistes

  • comenceu cada línia amb un asterisc * per crear una llista
  • utilitzeu # per a llistes numerades
    1. podeu crear subllistes
    2. els ítems de llistes subsegüents han de començar amb la mateixa combinació d'* i #

Hiperenllaços

  • escriviu un MotWiki dins del text per crear un nou hiperenllaç
  • poseu uns mots [entre claudàtors] per crear un hiperenllaç que no sigui un MotWiki
  • qualsevol adreça web vàlida (que comenci amb http:/) com ara http://www.google.com/ es convertirà en un enllaç automàticament
  • poseu una adreça web o un WikiLink entre claudàtors i assigneu-li un títol entre cometes o amb el caràcter |
    • [títol | http://www.google.com]
    • [MotWiki "títol"] or ["títol" WikiLink]
  • si no voleu que un MotWiki o una adreça web http:// (o [quelcom] que vagi entre claudàtors) es converteixi en un enllaç, poseu davant un signe d'admiració o una titlla:
    • !NoEnllaç, ~NoEnllaç/li>
    • ![no enllaç], !http://noenllaç.org/

Taules amb |

| poseu el text | entre caràcters | de traç vertical |
| per construir | una estructura de taula |
| els navegadors normalment | ignoren les cel·les | que falten |

Si us plau deixeu sempre una línia en blanc abans de la taula i una altra després per separar-la dels paràgrafs adjacents.

Imatges

  • per incloure una imatge dins d'una pàgina, poseu la seva adreça entre claudàtors: [http://www.example.com/pics/image.png]
  • alternativament podeu fer servir la funció de càrrega d'imatges
