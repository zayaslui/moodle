Richtext HTML редактор является программой, напоминающей MS WORD, но встроенной в Ваш браузер.

В дополнении к форматированию текста, редактор предлагает ряд дополнительных возможностей.

Копирование текста через буфер обмена из других программ

    Вы можете вставить текст из других windows-приложений, при этом формат текста будет сохранен (Control-C - для копирования и Control-V - для вставки).

Вставка картинок

    Если Вы хотите вставить картинку, находящуюся на web-сайте и доступную через URL, воспользуйтесь кнопкой "Вставить картинку"("Insert Image").

Вставка таблиц

    Вы можете вставить таблицу в текст, используя кнопку "Вставить таблицу" ("Insert Table").

Вставка ссылки

    Для того, чтобы создать web-ссылку наберите текст, выделите его, нажмите на кнопку "Ссылка" ("Link") на панели инструментов и наберите URL, на который будет указывать данный текст.

Вставка "Смайлов" (Эмоции)

    Для вставки маленьких иконок, выражающих эмоции, в текст, нажмите на иконку со "смайлом" на панели инструментов. Диалоговое окно позволит Вам выбрать нужную картинку (альтернативный способ рисования - написать текстовый эквивалент, который позднее автоматически заменится графическим аналогом).
