Помощь в написании HTML

При написании HTML на сайте, Вы можете использовать любые HTML-теги, для создания нужных Вам эффектов.

Обычно Ваш код будет размещен в ячейке таблицы, таким образом:

  • не нужно использовать теги <HEAD> или <BODY>;
  • будьте внимательны с использованием тега </TABLE>, который может сломать структуру страницы сайта.
