Вопросы со случайными вариантами ответов

После некоторого вступления, студенту предоставляется несколько "подвопросов" и несколько вариантов ответов. Для каждого "подвопроса" только один правильный ответ.

Студент должен выбрать правильный ответ, для каждого вопроса.

Каждый правильно ответ на "подвопрос" вносит одинаковую долю в окончательную оценку вопроса.

Вопросы и ответы случайным образом берутся из раздела "Вопросы, с ответами в виде слова или фразы" в текущей категории. Каждая попытка пройти тест будет содержать разные вопросы.
