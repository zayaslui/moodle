Überblick

  1. Eine Lektion besteht aus mehreren normalen Seiten und optionalen Verzweigungsseiten.
  2. Eine Seite stellt den Lerninhalt dar und endet normalerweise mit einer Frage.
  3. Jeder Seite und ihrer Frage ist im Normalfall eine Reihe von Antworten zugeordnet.
  4. Jeder Antwort kann kurzer Text zugeordnet werden, der angezeigt wird, wenn diese Antwort ausgewählt wird. Wir nennen diesen Text Rückantwort bzw. Reaktion.
  5. Zu jeder Antwort gehört ebenso ein Sprung. Der Sprung kann auf diese Seite oder die nächste Seite in der Lektion (sog. relativer Sprung) oder auf eine beliebige andere Seite in einer Lektion oder das Ende der Lektion verweisen (absoluter Sprung).
  6. In der Grundeinstellung verweist der Sprung der ersten Antwort einer Frage immer auf die folgende Seite der Lektion. Bei einer falschen Antwort springt das Programm auf den Anfang der aktuellen Seite zurück. Dadurch wird dem Teilnehmer immer wieder die gleiche Seite beim Auswählen der ersten Antwort gezeigt.
  7. Die nächste Seite ergibt sich aus der logischen Reihenfolge der Lektion. Das ist die Reihenfolge der Seiten, wie sie auch der/die Trainer/in sieht. Diese Reihenfolge kann innerhalb der Lektion durch Verschieben geändert werden.
  8. Die Lektion hat darüber hinaus eine Ablauffolge. Das ist die Reihenfolge in der der/die Teilnehmer/in die Seiten der Lektion sieht. Sie wird durch die Sprünge der verschiedenen Antworten festgelegt. und kann sich von der logischen Reihenfolge stark unterscheiden. Die Ablauffolge kann von der/dem Trainer/in überprüft werden ("Navigation überprüfen").
  9. Die Antworten werden für die Teilnehmer immer gemischt. Die erste Antwort in der Übersicht des/der Trainers/in ist nicht zwangsläufig die erste angezeigte Antwort für den/die Teilnehmer/in. Es werden immer alle vorgegebenen Antworten gezeigt - jedoch in unterschiedlicher Reihenfolge.
  10. Die Zahl der Antworten kann von Seite zu Seite variieren. Zum Beispiel ist es möglich, dass eine Frage mit Richtig oder Falsch beantwortet werden muß und die nächste eine richtige Antwort und drei falsche Antworten aufweist.
  11. Es ist auch möglich Seiten ohne Fragen und Antworten anzulegen. Die Teilnehmer/innen erhalten dann einen Fortsetzen Knopf anstelle von Antwortvorgaben.
  12. Wenn Sie die Bearbeitung der Lektion bewerten wollen, führen richtige Antworten zu einer Seite, die in der logischen Reihenfolge weiter hinten stehen. Falsche Antworten verweisen dann auf den Anfang der gleichen Seite oder auf eine frühere Seite zurück, um den Stoff zu wiederholen. Wenn Sie die Sprünge nicht verändern, ist die erste Antwort die richtige Antwort, alle anderen Antworten sind falsch.
  13. Fragen können jedoch mehr als eine richtige Antwort haben. Wenn zum Beispiel zwei Antworten einen Sprung zur nächsten Seite enthalten, geht das Moodle davon aus, dass beide Antworten richtige Antworten sind. (Die Teilnehmer/innen werden dann zur gleichen Seite geführt - unabhängig davon welche der beiden richtigen Antworten sie angegeben haben. Die Rückantworten zu den abgegebenen Antworten können jedoch unterschiedlich definiert werden.)
  14. In der Trainer/innen-Ansicht werden die richtigen Antworten unterstrichen angezeigt.
  15. Verzweigungsseiten sind einfache Seiten mit einer Anzahl von Links auf andere Seiten innerhalb der Lektion. Typischerweise könnte eine Lektion mit einer Verzweigungsseite starten, die das Inhaltsverzeichnis der Lektion enthält.
  16. Jeder Link in einer Verzweigungsseite besteht aus zwei Angaben, der Beschreibung und dem Titel der Seite, wo hingesprungen wird.
  17. Eine Verzweigungsseite unterteilt eine Lektion effektiv in verschiedene Zweige (Bereiche). Jeder Zweig kann eine Anzahl von Seiten enthalten (die sie z.B. all auf das gleiche Thema beziehen). Das ENde einer Verzweigung wird überlicherweise mit einem Verzweigungssende markiert. Diese ist eine spezielle Seite, die standardmässig den Teilnehmer auf den vorherige Verzweigungsseite zurückführt. (Das Rücksprungziel kann aber auch bei Verlangen durch Bearbeiten der Seite geändert werden.)
  18. Sie können mehr als eine Verzweigungsseite innerhalb einer Lektion benutzen. Z.B. können eine Lektion so strukturieren, dass spezielle Punkte Unterverzweigungen von Hauptzweigen sind.
  19. Es ist wichtig, dass Sie dem Teilnehmer die Möglichkeit geben die Lektion zu beenden. Dies könenn Sie tun, indem Sie einen 'Lektion beenden'-Verweis in der Haupt-Verzweigungsseite anlegen. Dieser Verweis springt dann zur (imaginären) "Ende der Lektion"-Seite. Eine andere Möglichkeit ist es, den letzten Zweig einer Lektion (wobei der 'letzte' in der logischen Reihenfolge gemeint ist) einfach zum Ende der Lektion führen zu lassen, d.h. dieser Zweig wird dann nicht durch eine Zweigende-Seite beendet.
  20. Wenn eine Lektion ein oder mehrere Verzweigungsseiten beinhaltet, sollten Sie unbedingt die Einstellung 'Minimale Anzahl an Fragen' auf einen vernünftigen Wert setzen. Dies setzt eine untere Grenze bei der Anzahl von Seiten, mit der die Note berechnet wird. Ohne diesen Wert könnte ein Student einen einzelnen Fragenzweig besuchen, alle Fragen richtig beantworten und dann die Lektion mit der Bestbenotung verlassen.
  21. Weiterhin hat ein Teilnehmer die Möglichkeit einen Fragenzweig mehrmals zu bearbeiten. Trotzdem wird die Note aber nur aus der Anzahl an unterschiedlichen und beantworteten Fragen berechnet. Mehrfach also den gleichen Fragenzweig zu beantworten, erhöht also nicht die Note. (Tatsächlich ist das Gegenteil der Fall - die Note wird gemindert, da die wiederholt gesehene Anzahl von Seiten in die Benotung mit einfliesst.) Um dem Teilnehmer eine vernünftige Übersicht seines Fortschritts innerhalb der Lektion zu geben, wird auf jeder Verzweigungsseite angezeigt, wieviele Fragen korrekt beantwortet wurden, wieviele Seiten angeschaut wurden und die aktuelle Note.
  22. Das Ende der Lektion ist erreicht, wenn die Seite, die am logischen Ende der Lektion steht, bearbeitet wurde. Der/die Teilnehmer/in erhält dann einen Glückwunsch und eine Bewertung. Die Bewertung (Zahl der als richtig ermittelten Antworten/Zahl der bearbeiteten Seiten) wird für die einzelne Lektion ermittelt.
  23. Wenn die Bearbeitung der Lektion nicht abgeschlossen wurde, kann der/die Teilnehmer/in entscheiden ob er/sie beim nächsten Start von vorne anfängt oder an der Stelle fortsetzt an der er/sie die Bearbeitung mit der letzten richtigen Antwort abgebrochen hat.
  24. Wenn die Wiederholung der Bearbeitung möglich ist, kann diese so lange erfolgen, bis die höchste mögliche Punktzahl erreicht wurde.
