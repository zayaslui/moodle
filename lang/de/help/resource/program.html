Programm Ressourcen

Eine Programm-Ressource ist eine externe Ressource, die von Moddle Daten übernehmen kann.

Die Daten werden per HTTP GET (als Teil der URL) übergeben und beinhalten:

  • extern_nav: die URL von der aus die Ressource aufgerufen wurde
  • extern_usr: den Usernamen des aktuellen Benutzers
  • extern_nam: den vollen Name des aktuellen Benutzers
  • extern_tim: die aktulle Zeit
  • extern_pwd: ein eindeutiges Passwort

All dies wird in nächster Zukunft überarbeitet werden.
