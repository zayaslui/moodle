Dimensione massima file inviati

Questa impostazione determina la dimensione massima dei file che gli studenti possono inviare. L'impostazione influenza solo i file inviati nell'ambito del corso ed è a sua volta limitata nel suo valore massimo dalla analoga impostazione disponibile a livello di sito per gli Amministratori del sistema.

E' inoltre possibile limitare ulteriormente la dimensione massima dei file inviati per ciascuna attività del corso.
